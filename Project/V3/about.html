About Page

Greetings fellow explorers! My name is Truman and I present to you the beginnings of the Boldly Going travel blog. This is being made as a part of my CIS155 course. Today's date is 25APR21. This is version 1 of my site. A list of changes will be available below.

Available on index.html is just the navigation console to be used as site navigation. There is also a picture which is just a placeholder for a future image that will be involved in navigation from the home page. Available on about.html is a description of the site and credits. Available on systems.html is a few images of systems visisted and page navigation.

No Man's Sky is a space themed game set in a fictional universe. It procedurally generated some 18 quintillion planets in this galaxy along with everything on those planets. Pretty cool stuff!
